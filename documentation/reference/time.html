--- title: Time Documentation layout: documentation --- Summary: The hardware clock is not present on all devices, but time/date mathematics is available on all platforms.

Time

convertTimeFromTicks

Convert from ticks in seconds to time Epoch is January 1st, 1997 (Wednesday)

Inputs

DE

Upper word of tick value

HL

Lower word of tick value

Outputs

A

Day of the week, from 0-6 with 0 being sunday

IX

Current year

C

Current minute, from 0-59

B

Current hour, from 0-23

E

Garbage

D

Current second, from 0-59

H

Current day, from 0-30

L

Current month, from 0-11

getClock

Sets the internal clock.

Inputs

A

Preserved on success, error code on failure

Z

Set on success, reset on failure

DE

Upper word of the 32-bit tick value

HL

Lower word of the 32-bit tick value

None ;; Outputs

setClock

Sets the internal clock.

Inputs

DE

Upper word of a 32-bit tick value

HL

Lower word of a 32-bit tick value

Outputs

A

Preserved on success, error code on failure

Z

Set on success, reset on failure
