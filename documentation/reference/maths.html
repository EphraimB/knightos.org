icos View Source

Returns the cosine of the given angle.

Inputs

A

angle

Outputs

A

sine of the angle, scaled to 64

Notes

To fit assembly, both the input and output have an uncommon format : the input angle has a period of 256, and the outputted value will be in the range [-63, 63] instead of [-1,1].
