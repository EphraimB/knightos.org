Miscellaneous Functions

add16To32

Performs ACIX = ACIX + DE

compareStrings

Determines if two strings are equal.

Inputs

HL: String pointer

DE: String pointer

Outputs

Z: Set if equal, reset if not equal

cpDEBC

Compares DE to BC.

Output

Same as z80 CP instruction.

cpHLBC

Compares HL to BC.

Output

Same as z80 CP instruction.

cpHLDE

Compares HL to DE.

Output

Same as z80 CP instruction.

DEMulA

Performs `HL = DE * A"

div32By16

Performs ACIX = ACIX / DE

Outputs

As described above, and:

HL: Remainder

B: 0

divACByDE

Performs AC = AC / DE

Outputs

As described above, and:

HL: Remainder

divHLByC

Performs HL = HL / C

Outputs

As described above, and:

A: Remainder

getBatteryLevel

Determines the approximate battery level.

Outputs

B: Battery level

Notes

B is a value from 0 to 4, where 0 is critical and 4 is full.

getBootCodeVersionString

Gets the version string from the device's boot code.

Outputs

HL: String pointer

Notes

This allocates memory to hold the string. Deallocate it with [[free]] when you no longer need it.

hexToA

Converts a hexadecimal string to a number.

Inputs

HL: String pointer

Outputs

A: Value

hexToHL

Converts a hexadecimal string to a number.

Inputs

HL: String pointer

Outputs

HL: Value

stringCopy

Copies a string.

Inputs

HL: String pointer

DE: Destination

stringLength

Determines the length of a zero delimited string.

Inputs

HL: String pointer

Outputs

BC: String length

sub16From32

Performs ACIX = ACIX - DE
