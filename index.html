Open Source

KnightOS is the first open-source operating system for Texas Instruments calculators. It aims to be a Unix-like system that runs the average z80 calculator, with multitasking and a proper filesystem and other features familiar to users of conventional PC operating systems. The end goal is to also provide a great operating system suitable for use in the classroom.

Programmers, head over to the documentation. Users, hang in there while we work on fleshing out the system for you. If you'd like, you can come join us on IRC to chat about the project and make sure you like where it's going. We need your direction!
