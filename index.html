KnightOS An open source OS for TI calculators

KnightOS is an open-source operating system for TI calculators. It completely replaces the stock TI-OS, and offers loads of great features for programmers and users alike:

Multitasking
Run up to 32 programs simultaneously
Cross Platform
Run your programs on 9 different calculators - including the new Color Silver Edition
Memory Safe
No more panicked RAM resets. Crashes do not affect user data.
More Memory
KnightOS programs may use 31K of RAM. TI-OS offers 24K under ideal conditions
Storage Space
KnightOS is compact, leaving plenty of space for your files
Open Source
You're freely able to browse, change, and improve the source code

Easy-to-use graphical interface

Runs several programs at once

Extensible with third party libraries

KnightOS supports the TI-73, TI-83+, TI-83+ Silver Edition, TI-84+, TI-84+ Silver Edition, TI-84+ Color Silver Edition, and the French variations on these devices.

KnightOS for Users

Sorry! KnightOS isn't quite ready for everyone. Currently, only technically proficient users will realistically be able to use the system. You can still join us on IRC to keep an eye on our progress and share your ideas.

KnightOS for Programmers

KnightOS will feel like a welcome change for programmers used to TI-OS. Coding for KnightOS feels like coding for Unix. You've got full blown memory management, a tree-based filesytem (one that follows the Filesystem Hierarchy Standard), and more "Safe RAM" than you can shake a stick at. KnightOS is also completely open source, which means that you can get involved in the development. The source code is availble online and you're encouraged to play around with it. If you see something you'd rather be different, or features you want, you can join us on IRC and help build it yourself!

KnightOS is also painstakingly documented, so you'll always know how it works. To get started writing programs for KnightOS, go here.
